## REQUIREMENTS ##

You need to have access to facebook dinov3 models in order to use Meshflow

Request the access here (https://ai.meta.com/resources/models-and-libraries/dinov3-downloads/)

Download dinov3_vitl16 model and save it in ComfyUI/models/facebook/ folder
